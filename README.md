# 📦ProVision Inventory (Inventory Management System)

### 🔗 Live Website: [https://inventory-management-sys-8bd31.web.app/](https://inventory-management-sys-8bd31.web.app/)

A robust and user-friendly inventory management system that helps businesses manage shops, users, and product inventory efficiently with role-based access control. This full-stack MERN application supports CRUD operations, user authentication, secure routing, and modern UI design.

---

## ✨ Features

### 🔐 Role-Based Access Control
- **Admin** can manage all users, shops, and inventories.
- **Shop Manager** can manage their own shop’s inventory.
- **Normal Users** have limited access to view features.

### 📋 Inventory Management
- Add, update, or delete inventory items.
- Filter items by shop, quantity, category, or name.
- Real-time updates to inventory and shop data.

### 🏪 Shop Management
- Add and manage multiple shops.
- Assign users to shops.
- View individual shop inventory.

### 👥 User Management
- Register/login with email and password.
- Role assignment and access restriction.
- Secure private routes with route guards.

---

## 🛠️ Tech Stack

### 🔹 Frontend
- **React JS** – Component-based UI
- **React Router DOM** – Client-side routing
- **Tailwind CSS + DaisyUI** – Modern, utility-first styling
- **Axios** – API requests
- **React Hook Form** – Form handling and validation
- **React Query (TanStack Query v5)** – Efficient data fetching and caching
- **AOS Animation** – Scroll animations

### 🔹 Backend
- **Node.js + Express.js** – RESTful API
- **MongoDB** – NoSQL database for storing users, shops, and inventory
- **JWT** – Authentication and authorization

### 🔹 Hosting & Auth
- **Firebase Authentication** – Login, register, and protect routes
- **Firebase Hosting** – Fast and secure app hosting
- **Vercel** – Backend deployment
